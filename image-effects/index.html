Image effects

How to use ShareX image effects?

  1. Download ".sxie" file below by clicking an effect of your choice.
  2. Double click the downloaded file. It will bring image effects window and ask if you want to enable image effects. Click "Yes" button.
  3. Feel free to edit the effect if you want or close the window.

Every time you take a screenshot, the image effect will be automatically applied to the screenshot.

To disable image effects, uncheck "Add image effects" in main window "After capture tasks" menu.

If you want to preserve transparency while copying image then make sure this option is enabled: "Application settings window -> Advanced tab -> UseAlternativeClipboardCopyImage"
